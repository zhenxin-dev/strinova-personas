# 来源总表

这份表用于登记仓库里每个文件应去哪里复核。

它记录的是 `来源登记状态`，不是“今天已经逐页重新核对过所有外部页面”的意思。

## 来源优先级

1. 角色主词条
2. 语音台词、誓约、羁绊剧情、初识剧情、卡丘通讯、角色设定页
3. 技能、武器、时装、玩法页、阵营页、地图页
4. 社区整理页
5. 保守推断

## 状态说明

- `已登记`：文件内已经给出详细来源位置，后续复核时有明确入口
- `规则文件`：仓库协作或使用规则，不直接承载官方设定事实
- `模板文件`：结构模板，依据仓库现有角色文件总结而来
- `示例文件`：组合示例，不新增官方设定
- `工具文件`：校验仓库结构，不直接承载官方设定
