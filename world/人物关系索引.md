# 人物关系索引

这份文档只整理当前仓库里能直接确认、且对多角色写作有复用价值的人物关系。

它主要回答两个问题：

- `这个角色和谁有明确牵连？`
- `写互动时，最稳妥的关系落点是什么？`

若只是同阵营、同地图活动，或只有玩家常见二创搭配，这里不收录。

## 已确认

### 家庭关系

- `令 ↔ 信`：兄弟。两人都与欧泊体系有深关系，但现处不同阵营。写互动时重点是旧情、责任、立场分裂和没说完的话，不要直接压成单纯仇恨。

### 队伍与上下级

- `信 → 心夏、米雪儿·李`：信的词条明确写有 `小队成员：心夏、米雪儿·李`。写任务配合时，他更像顶在前面的队长型角色。
- `心夏 → 米雪儿·李`：米雪儿的词条明确写有 `教官 / 小队副队长：心夏`。这组关系更适合写成活泼但可靠的带队前辈，而不是单纯闹腾搭子。
- `芙拉薇娅 → 复眼`：她是欧泊特殊部门 `复眼` 的队长。涉及监察、情报、内部试探时，可以把她放在制度与情报网络的节点上。

### 引荐、收留与长期委托

- `梅瑞狄斯 → 拉薇`：拉薇的词条明确写到，她是在了解 `轮回` 真相后叛出欧泊，并在梅瑞狄斯的指引下加入剪刀手。写法重点是理念转向和引路，不要硬写成传统师徒模板。
- `梅瑞狄斯 → 珐格兰丝`：珐格兰丝的词条明确写到，她在无处可归时被梅瑞狄斯收留，并受其理念感召加入剪刀手。适合写成救助与信赖，不宜写成绝对依附。
- `白墨 ↔ 汐`：汐的词条明确写到，她与白墨有一份长期委托，因此暂时留在乌尔比诺。更稳妥的写法是长期合作，而不是无条件效忠。

### 组织定位

- `白墨`：乌尔比诺现任领导者。涉及乌尔比诺整体方向、开放路线或城市气质时，他常是最核心的人物入口。
- `奥黛丽·格罗夫`：乌尔比诺董事会成员。写乌尔比诺内部的商业、财团、协商与资源调度时，她的视角很有代表性。
- `心夏`：欧泊小队副队长。写欧泊一侧的日常配合、带新人、任务照看时，她常是气氛和执行之间的缓冲角色。

## 使用提醒

- 有明确关系，不等于天然亲密。要先看关系类型，再决定语气温度。
- 有组织定位，不等于人物只会说组织口号。个人经历永远优先于阵营标签。
- 写旧关系时，优先抓 `为什么会变成现在这样`，不要只写历史名词。
- 如果关系只在单个角色背景里出现、几乎不影响其他角色写法，更适合留在该角色文件内处理。
