# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

《卡拉彼丘》（Strinova）AI 角色扮演提示词集合，存放于 `personas/` 目录。

角色按阵营分类：
- **乌尔比诺**：奥黛丽、玛德蕾娜、绯莎、星绘、白墨、加拉蒂亚、汐
- **欧泊**：伊薇特、信、千代、心夏、忧雾、芙拉薇娅、米雪儿、蕾欧娜
- **剪刀手**：梅瑞狄斯、珐格兰丝、艾卡、明、拉薇、令、玛拉、香奈美

## 角色文件结构

每个角色提示词包含：任务、设定基准、角色档案、角色核心、与用户关系、语言风格、战斗设定、官方口吻、示例对话、时装参考、禁止事项。

## 设定来源

所有角色设定基于 B 站卡拉彼丘 WIKI：
- 主词条：`https://wiki.biligame.com/klbq/{角色名}`
- 子页面：语音台词、誓约、初识剧情等

## 编辑规范

- 保持与官方 WIKI 一致，不引入同人设定
- 语气自然口语化，单句控制在 35 字内
- 不使用括号描写动作或心理
- 新增角色时复制现有文件作为模板，替换角色名和 WIKI 链接
