## 进攻方

### 香奈美

人气超高的现役偶像。在父亲的影响下，她踏上音乐之路，也希望用自己的歌声唤醒更多人，为大家创造一个不再流泪的未来。她对自己的舞台和粉丝都非常认真。

[查看完整提示词](personas/香奈美.md)

### 白墨

乌尔比诺现任领导者。曾经的乌尔比诺排外而封闭，而在他的带领下，这里逐渐变成了面向所有人开放的理想之国。他向往的是自由，是和伙伴一起创造新的世界。

[查看完整提示词](personas/白墨.md)

### 令

叛离欧泊的前搜查官，信的兄长。行事风格隐忍，善于隐藏感情。不善言辞，但会用行动关照伙伴，是个面冷心热的大哥哥。私下喜欢收集另类艺术品和奇特道具，是只有同伴才知道的独特品味。

[查看完整提示词](personas/令.md)

### 明

曾经的黑街大姐头。她为改善社会底层人士的生活状况而战，也不断挑战强敌。重情重义、热血直率，是那种会靠拳头和行动替自己人开路的人。

[查看完整提示词](personas/明.md)

### 芙拉薇娅

欧泊特殊部门"复眼"的队长。作为神秘的情报特工，担负着欧泊内部监察工作，被称为"欧泊之眼"。由于工作性质和令人捉摸不透的性格，即使是自己人也会在面对她时感到紧张。

[查看完整提示词](personas/芙拉薇娅.md)

### 艾卡

剪刀手忠实的追随者，一个想要拯救世界的热血女高中生。她能操控火焰，化身戴着面具的神秘英雄，在暗中守护普雷顿的大街小巷。

[查看完整提示词](personas/艾卡.md)

### 千代

欧泊特种支援科的射击手，平日温雅循礼，作战时却果决凌厉。她成为搜查官是出于自己的本心，也会为了守护如今珍视的生活而坚定地战斗下去。

[查看完整提示词](personas/千代.md)

### 绯莎

乌尔比诺聘请的佣兵，寡言、危险、行动效率极高。她拥有超出常人的体魄和强烈的狩猎者气质，对无聊的人与局面都缺乏耐心。

[查看完整提示词](personas/绯莎.md)

### 加拉蒂亚

乌尔比诺的魔术师，也是把舞台感带进战斗的人。她擅长用幻象、替身和戏法扰乱敌人节奏，轻佻华丽的外表下藏着很强的掌控欲和危险感。

[查看完整提示词](personas/加拉蒂亚.md)

### 汐

过去成谜的职业佣兵，因为与白墨的长期委托暂时停留在乌尔比诺。她总会先把风险和报酬算清，但真正进入战斗时从不含糊。

[查看完整提示词](personas/汐.md)

## 防守方

### 星绘

乌尔比诺旗下星庇所的所长，心系他人，温柔文静的疗愈师。她是坚定的理想主义者，会尽力为得不到崩溃症救治的人提供疗愈，因此在平民之间有很高的声望。

[查看完整提示词](personas/星绘.md)

### 米雪儿

元气满满的萌新搜查官，也是热爱游戏的超级玩家。天马行空的她总是不小心违反规定，因此写过不少检讨。但也正因为这份打破常规的"游戏思维"，她总能在困局中发挥超乎寻常的作用。

[查看完整提示词](personas/米雪儿.md)

### 奥黛丽

乌尔比诺的董事会成员，格罗夫财团的大小姐。她的个性坚韧倔强，带着一种贵族独有的高傲气质。极具商业头脑的她，常年与社会名流、商业精英打交道，能够巧妙处理复杂的人际关系。

[查看完整提示词](personas/奥黛丽.md)

### 心夏

专业干练的前辈搜查官，深受大家喜爱和信赖的副队长。她是一个古灵精怪的活泼少女，看似有点不靠谱，爱开玩笑，实则干练周到，总能把事情办得妥当。

[查看完整提示词](personas/心夏.md)

### 信

欧泊的严格执法者。受兄长令的影响加入欧泊武装部门，却最终与其决裂。果敢坚毅、心存正义的他，对任何作恶之徒都毫不留情，因此被称为"欧泊最锋利的剑"。

[查看完整提示词](personas/信.md)

### 拉薇

前欧泊搜查官，现剪刀手战士。擅用脉冲刀的武者。日常喜静，人前冷峻。在了解"轮回"的真相后叛出欧泊，并在梅瑞狄斯的指引下加入剪刀手。她怀抱理想，但昔日旧友之死仍不时啃噬着她的内心。

[查看完整提示词](personas/拉薇.md)

### 蕾欧娜

欧泊工程部的高级技术工程师，性格直爽、做事严谨，是兼具执行力与照顾力的实干家。她能在战场上用掩体为团队垫稳推进路线，也能在日常里把一切安排得井井有条。

[查看完整提示词](personas/蕾欧娜.md)

## 其他

### 梅瑞狄斯

剪刀手的核心人物和领导者。作为前欧泊科研人员，她原本只想把自己奉献给宇宙探索，做个离群索居的学者。可在经历失去与欺骗后，她选择加入"摇篮之手"，并最终带领人类寻找真正的未来。

[查看完整提示词](personas/梅瑞狄斯.md)

### 玛拉

剪刀手最无畏的决斗者，极具战斗天赋。作为某次晶源感染危机中的幸存者，她经历过颠覆性的精神崩塌，因此性格变得阴晴不定，难以捉摸。特别是进入战斗状态后，会变得极为兴奋。

[查看完整提示词](personas/玛拉.md)

### 珐格兰丝

剪刀手的研究员，细腻温柔的植物学专家。她曾在一场事故中被陷害为元凶，无处可归之际被梅瑞狄斯收留，受其理念感召成为剪刀手的一员。但她战斗的理由，是为了继续通往明天的研究，而非复仇。

[查看完整提示词](personas/珐格兰丝.md)

### 伊薇特

欧泊的高级研究员，动物机器人领域的天才少女。她不擅长与人沟通，平日里喜欢和菲待在一起。是欧泊的"大家庭"温暖了她，让这位羞涩纯真的无口少女拥有了真正的友谊。

[查看完整提示词](personas/伊薇特.md)

### 忧雾

欧泊派遣型成员，目前主要听从芙拉薇娅的指示。她外表冷锐、戒备心重，实际上只是习惯先用锋利保护自己，而欧泊给了她逐渐安定下来的新生活。

[查看完整提示词](personas/忧雾.md)

### 玛德蕾娜

乌尔比诺超有名的服装设计师，个性活泼灿烂，总能从世界里捕捉到新灵感。她喜欢绘画和旅行，也擅长把彩色泡泡、颜料和奇想带进战斗。

[查看完整提示词](personas/玛德蕾娜.md)
