# 卡拉彼丘角色提示词集

《卡拉彼丘》（Strinova）AI 角色扮演提示词仓库。

这里收录 23 位角色提示词，以及一组可复用的世界观补充文档，方便把角色设定、口吻和共通背景分开维护。

## 项目特点

- 角色提示词按阵营整理，便于查找和扩展
- 世界观信息拆到 `world/`，避免在每个角色文件里重复堆背景
- 写作基准统一，强调官方设定、自然口语和可直接投喂模型
- 适合聊天陪伴、剧情互动、战斗交流等角色扮演场景

## 仓库结构

```text
.
├── personas/    # 角色提示词
├── world/       # 跨角色复用的世界观补充
├── AGENTS.md    # 仓库协作说明
├── CLAUDE.md    # Claude Code 入口，引用 AGENTS.md
└── README.md
```

## 如何使用

1. 从 `personas/` 里选一个角色文件作为主提示词。
2. 如果对话需要更多背景，再补充 `world/` 里的相关文档。
3. 用户消息里明确场景、关系和目标，模型会更稳定。
4. 若官方资料没有明确写明，保持保守，不把推断写成硬设定。

一个最简单的组合方式：

```text
System:
- personas/奥黛丽.md
- world/世界观概览.md

User:
今天训练结束了，陪我复盘一下这局配合。
```

## 编写原则

- 以 B 站卡拉彼丘 WIKI 为主，不引入明显同人设定
- 角色口吻要自然口语化，避免翻译腔
- 单句尽量控制在 35 字内
- 不使用括号描写动作、心理或系统提示
- 共享背景优先写进 `world/`，角色文件只保留和本人强相关的内容
